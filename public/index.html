📡 Welcome to the Hadlard Biometric Student API

This backend powers a biometric student data collection and attendance app.

Features

  • 📋 Collect student records (name, age, class, etc.)
  • 🏫 Track classes by school and grade
  • ✅ Designed for biometric attendance marking
  • 📤 Batch upload of student data from mobile app
  • 📦 Built with Express + MongoDB

📌 Available Endpoints

POST /api/students
GET /api/students
POST /api/students/batch
